# Projet-Big-Data-2026

Projet sur le patrimoine arboré de St-Quentin

## Nettoyer le dataset

Pour nettoyer le dataset initial, il faut exécuter le script main.r. Il appelle les différents fonctions du dossier R, et exporte le dataset nettotyé.

## Afficher les graphiques

Il faut exécuter le script visualisation.R dans le dossier exploration.

## Afficher les cartes

Il faut exécuter le script maps.R du dossier maps.
